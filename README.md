# CSV Data Analyzer 📊

## Description
A command-line data analysis tool built using Python’s built-in CSV module. It allows users to load, analyze, filter, and update structured data without external libraries.

## Features
- Load and display CSV data
- Statistical analysis (average, max, min)
- Filter by score and search by name
- Subject-wise grouping and analysis
- Add new data and save to CSV

## Tech Used
- Python
- CSV module
- File handling

## How to Run
python main.py
